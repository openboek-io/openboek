💡 {% if lang == 'en' %}What is a holding company?{% else %}Wat is een holding?{% endif %}

{% if lang == 'en' %} A holding BV is a separate company that owns your shares in your operating company (werk-BV). Instead of owning the operating BV directly, your holding owns it. {% else %} Een holding BV is een aparte vennootschap die jouw aandelen bezit in je werkmaatschappij (werk-BV). In plaats van de werk-BV direct te bezitten, bezit je holding die. {% endif %}

{% if lang == 'en' %}Why? Benefits of a holding structure:{% else %}Waarom? Voordelen van een holdingstructuur:{% endif %}

  • {% if lang == 'en' %}Profits flow up tax-free (deelnemingsvrijstelling / participation exemption){% else %}Winst stroomt belastingvrij omhoog (deelnemingsvrijstelling){% endif %}
  • {% if lang == 'en' %}Better asset protection — if the operating BV fails, the holding's assets are safe{% else %}Betere vermogensbescherming — als de werk-BV failliet gaat, is het vermogen in de holding veilig{% endif %}
  • {% if lang == 'en' %}You can defer dividend distributions{% else %}Je kunt dividenduitkeringen uitstellen{% endif %}

💡 {% if lang == 'en' %} A management fee is what your holding charges the operating BV for your services as director (DGA). This is standard practice and tax-efficient. {% else %} Een managementfee is wat je holding in rekening brengt bij de werk-BV voor jouw diensten als directeur (DGA). Dit is gebruikelijk en fiscaal efficiënt. {% endif %}
